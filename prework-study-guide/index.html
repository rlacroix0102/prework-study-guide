HTML

  • The head element contains information about the webpage
  • The body element represents visible content shown to the user

CSS

  • 3 style sheets: inline, internal, and external
  • inline: styles directly into the HTML file
  • inline: is also in the HTML file but is only in the head element
  • external: most common/preferred, allows devs to keep all CSS rules in a seperate file, makes design changes easier
  • you use a link element placed in the HTML's head element
  • link rel="stylesheet" href="./assets/style.css" should look like this with brackets around it
  • The padding property is often confused with margin, but it is not the same thing. The padding property adds space around the content inside an element. When you increase padding, you are actually making the entire element bigger! That is because the padding, while transparent, is located inside the element, whereas the margin is the transparent buffer outside an element
  • * symbol gives all elements on a page the style properties
  • The period preceding card designates this as a class selector. Any HTML element assigned to the class, card, will gain these property assignments from the CSS
  • The border property assigns a border
  • The margin property assigns 40px to the top and bottom, and the auto value to the left and right. ex: margin: 40px auto;

JavaScript

  • variable: is simply a named container that allows us to store data in our code.
  • use the keyword const to create a variable
  • ex: const topics = "CSS, JavaScript";
  • The variablke is called topics and the string includes the topics "CSS, JavaScript
  • The string is always surrounded by quotation marks
  • JavaScript lines always end with a semicolon
  • Use console.log to add something to the console
  • ex: console.log(topics);
  • Link JS files in the body element
  • Array: is a single variable that is used to hold a group of data
  • Ty[ically used to store data that is related ex: shapes
  • To create an array, we declare a variable in the same way that we declared a variable with only one data item. That is, we use const and then give the variable a name. However, instead of assigning the variable to a single value, we assign the variable to a group of values.
  • ex: const shapes = ["triangle", "square", "pentagon", "circle"];
  • Arrays can also store multiple types of data ex: information about a student
  • ex: const studentInfo = ["Lu", 54, true];
  • Lu is in quotation marks because it is a string, 54 is a number (the students age), and true is a Boolean value to indetify enrollment status so they do not need to be in quotations
  • To work with individual data items inside an array, we need to have one more piece of information: the location of the data item in the array. This location is identified with a unique number called an index.
  • The first item in an array has an index of 0 and goes up by 1
  • ex: shapes[0]; this would access the first item in an array of shapes
  • To see it in action you would do console.log(shapes[0]);
  • If we had an array that looked like: const shapes = ["triangle", "square", "pentagon", "circle"];
  • You could keep typing console.log(shapes[0]); console.log(shapes[1]); and so on but it is more efficient to use a loop
  • For this example it is best to use a for loop which contains 3 important statements
  • The first statement determines the starting point for our loop. Since we want to start at the beginning of the array, we declare a variable x and give it a value of 0 (because the first index of an array is always 0) as shown in: let x = 0;
  • The second statement is the condition. As long as the number in the variable x is less than the length of the array, the loop will keep running. This will allow the loop to execute one time for each item in the array. We use the length built-in array property on our shapes array to express that condition, as shown in the following line of code: x < shapes.length;
  • The final statement is what allows the array to iterate over each item. Since indices follow a predictable pattern, we know the number increases by 1 each time. The code x++ is a shortcut way of writing x+1. Now every time the code loops, x will increase by 1, as shown in the following line of code: x++
  • A for loop starts with for and should look like: for(let x = 0; x < shapes.length; x++) { // code block}
  • Finally, we put the code that we want to be executed for each loop inside curly brackets. In our case, we want each of the items in the shapes array to be logged in the console. In place of an actual index number, however, we can add the variable x from our for loop
  • The shapes ex. above would look like:
  • const shapes = ["triangle", "square", "pentagon", "circle"];
  • for(let x = 0; x < shapes.length; x++) { console.log(shapes[x]); }
